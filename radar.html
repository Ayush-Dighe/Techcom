Objective -
My project rader detector helps to detect any object in the certain range, if it detects any object the limited range it reports in the processing radar. This project is a miniature form of the radar. It is used on navy, army, etc.

Materials used Set of jumping wires, 1 servo motor, 1 ultrasonic sensor and arduino uno
