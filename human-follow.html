Arduino Human Following Robot

Objective - This robot detects movements which it senses from both its sensors: Infrarede sensor and ultrasonic sensor ,this robot is made using Arduino, an open source platform which help us to build various things. The main motive of this robot is to follow humans.

Materials used : TT Gear Motor, Acrylic sheet ,Glue gun, Rubber Wheels, Arduino Uno, Motor drives, Server motor ,Infrared sensor, Battery and holder , Ultrasonic sensor

CLASS: 7-A
